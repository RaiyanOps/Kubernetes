Restart control-plane components (CRITICAL)

Note : you have multinode kubernets cluster ( 3 control plane node )

After renewal, restart kubelet on ALL 3 masters, one by one.

On Master-1

* sudo systemctl restart kubelet

Wait ~30–60 seconds, then check:

* kubectl get nodes

Repeat on Master-2 and Master-3

* sudo systemctl restart kubelet


Do NOT restart all masters at once — keep quorum.

Refresh kubeconfig files (COMMON HA ISSUE)

On ALL 3 masters, run:

* sudo kubeadm init phase kubeconfig all


Then fix kubectl access (each master if used):

mkdir -p $HOME/.kube

sudo chown $(id -u):$(id -g) $HOME/.kube/config


Test:

* kubectl get nodes
